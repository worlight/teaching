Windows 11 快捷鍵互動指南

您的終極鍵盤快捷鍵助手，由學校IT部門提供。

找不到相符的快捷鍵 😔

請嘗試使用其他關鍵字。

常用快捷鍵

Ctrl + C

複製選取的項目。

Ctrl + X

剪下選取的項目。

Ctrl + V

貼上複製或剪下的項目。

Ctrl + Z

復原上一個動作。

Ctrl + A

選取文件或視窗中的所有項目。

Alt + Tab

在開啟的應用程式之間切換。

Ctrl + Alt + Del

開啟工作管理員或安全性選項。

F5

重新整理作用中的視窗。

Windows 鍵快捷鍵

Win

開啟或關閉 [開始] 功能表。

Win + A

開啟 [快速設定] 面板。

Win + C

開啟 Microsoft Teams Chat。

Win + D

顯示及隱藏桌面。

Win + E

開啟檔案總管。

Win + G

開啟 Xbox Game Bar。

Win + H

啟動語音輸入。

Win + I

開啟 [設定]。

Win + K

開啟 [投射] 快速設定。

Win + L

鎖定您的電腦。

Win + M

將所有視窗最小化。

Win + N

開啟通知中心與行事曆。

Win + R

開啟 [執行] 對話方塊。

Win + S

開啟 [搜尋]。

Win + V

開啟剪貼簿歷程記錄。

Win + W

開啟 [小工具]。

Win + X

開啟 [快速連結] 功能表。

Win + Z

開啟貼齊版面配置。

Win + Tab

開啟工作檢視 (虛擬桌面)。

Win + Ctrl + D

新增虛擬桌面。

Win + Ctrl + 左右方向鍵

在虛擬桌面之間切換。

Win + Ctrl + F4

關閉使用中的虛擬桌面。

Win + Shift + S

開啟螢幕剪取工具。

Win + . (Period)

開啟 Emoji 面板。

Win + 方向鍵

貼齊視窗 (上:最大化, 左/右:貼齊半邊, 下:最小化)。

檔案總管

Ctrl + N

開啟新視窗。

Ctrl + T

開啟新索引標籤。

Ctrl + W

關閉作用中的索引標籤。

Ctrl + Shift + N

建立新資料夾。

Alt + P

顯示/隱藏預覽窗格。

F2

重新命名選取的項目。
